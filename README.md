# Криптография
Изучение исторических криптографических алгоритмов и протоколов.

## 01. Шифр Цезаря
В классическом шифре Цезаря используется латинский алфавит. Зашифрование происходит посимвольным смещением каждой буквы на три вперёд (A ~ D, B ~ E и так далее), а расшифрование смешением буквы на три назад. Строго говоря, это алгоритм шифрования без ключа, поэтому, такого понятие как взлом к нему не применимо (зная алгоритм, сразу применяем агоритмя расшифровки).

## 02. Обобщённый шифр Цезаря
Обобщённый шифр Цезаря использует произвольный алфавит и произвольное смещение. Для взлома необходимо найти такое свещение, чтобы из шифро-текста получить что-то осмысленное.

Алфавит A &ndash; произвольное конечное упорядоченное множество. Поэтому для работы с ними можно использовать взаимооднозначное отоброжение множеству классов вычетов по модулю равной размеру алфвита.

A ~ Zn
- A &ndash; алфавит;
- |A| = n &ndash; мошность алфавита;
- Zn &ndash; множество классов вычетов по модулю n.
